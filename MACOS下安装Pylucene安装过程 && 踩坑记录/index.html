    MACOS下安装Pylucene安装过程 && 踩坑记录

    安装JDK

    此处注意下JDK8，如果下最新版JDK12后续会出错

    安装JCC

    在http://lucene.apache.org/pylucene/下载安装包

    # https://mirrors.tuna.tsinghua.edu.cn/apache/lucene/pylucene/pylucene-7.6.0-src.tar.gz
cd pylucene-7.6.0
cd jcc
python3 setup.py install
#如果JDK装的不对此处会报错

    #python
#安装好之后，测试
import jcc

    安装pylucene

    安装ANT（依赖）

    brew install ant

    改Makefile

    #增加
ANT=/usr/local/bin/ant
PYTHON=python3
JCC=$(PYTHON) -m jcc.__main__ --shared
NUM_FILES=2

    编译

    make
#此处如果开多线程下一步会报错 make -j4不行
sudo make install

    #python
#安装好之后，测试
import lucene
